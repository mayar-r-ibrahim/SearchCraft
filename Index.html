Advanced Search Parameters

Term-Level Operators
Exact Phrase Operator

Finds pages containing this exact phrase in this exact order. Use quotes to search for specific phrases.

Example: "to be or not to be" finds pages with that exact Shakespeare quote
Example: "machine learning" finds pages with that exact phrase, not just both words separately
OR Operator

Finds pages containing any of the specified terms. Must use uppercase OR between terms.

Example: java OR python tutorial finds pages about either Java tutorials or Python tutorials
Example: cats OR dogs pets finds pages about either cats or dogs as pets
Exclude Operator

Excludes pages containing the specified terms from search results.

Example: jaguar speed -car finds information about jaguar (animal) speed, not Jaguar cars
Example: apple -fruit -computer finds pages about Apple Inc. but not about the fruit or computers
Asterisk Wildcard

Matches any number (zero or more) of characters. Used within quotes to find variations.

Example: "govern*s" finds governors, governments, governs
Example: "a * saved is a * earned" finds variations of the saying
Example: "how to * quora" finds "how to use quora", "how to post on quora", etc.

Question Mark Wildcard

Matches any single character at the position of the question mark.

Example: wom?n finds woman, women
Example: te?t finds text, test
Example: sm?th finds smith, smyth

Note: Google doesn't directly support the ? wildcard in regular search, but this helps you construct patterns for other search systems.

Preview: Enter a pattern to see preview
Brackets Wildcard

Matches any single character within the brackets. You can also use ranges like [a-z] or [0-9].

Example: b[ae]ll finds ball, bell
Example: gr[ae]y finds gray, grey
Example: [0-9] matches any single digit
Example: countr[yies] finds country or countries

Note: Google doesn't directly support the [] wildcard in regular search, but this helps you construct patterns for other search systems.

Site & URL Based Operators
Site Operator

Restricts results to a specific website or domain.

Example: site:github.com python API finds Python API information only on GitHub
Example: site:.gov climate report finds climate reports only on government websites
Example: site:edu machine learning finds machine learning content only on educational websites
Intitle Operator

Finds pages whose HTML title tag contains the specified word.

Example: intitle:"index of" database finds directory listings with "database" in the title
Example: intitle:resume software engineer finds resumes of software engineers
Inurl Operator

Finds pages whose URL contains the specified word.

Example: inurl:blog travel tips finds travel tip pages with "blog" in the URL
Example: inurl:download python finds Python download pages
File Type & Content Operators
Filetype Operator

Finds specific file types. Extremely powerful for research.

Example: filetype:pdf neural networks finds PDF documents about neural networks
Example: ext:ppt marketing strategy finds PowerPoint presentations about marketing strategy
Example: filetype:doc resume template finds Word document resume templates

Common filetypes: pdf, ppt, pptx, doc, docx, xls, xlsx, txt, rtf, zip

Related Operator

Finds sites similar to a given site.

Example: related:reddit.com finds forums similar to Reddit
Example: related:github.com finds code repository sites similar to GitHub
Example: related:amazon.com finds e-commerce sites similar to Amazon
