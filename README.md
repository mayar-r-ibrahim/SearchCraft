**üõ†Ô∏è Usage Guide**
          
  **Simple Search Mode**
          Enter your search terms in the main search box
          
          Click any search engine button for instant results
          
          Or use "Open All 12 Search Engines" to search everywhere
          
  **Advanced Search Mode**
          Build complex queries using these operators:
          
          Operator	Example	Description
          "exact phrase"	"machine learning"	Finds exact phrase matches
          OR	java OR python	Results containing either term
          -	jaguar -car	Excludes specific terms
          site:	site:github.com	Restricts to specific domain
          filetype:	filetype:pdf	Filters by file type
          intitle:	intitle:"index of"	Searches within page titles
          inurl:	inurl:download	Searches within URLs
          *	"govern*"	Wildcard for any characters
          Wildcard Search Types
          Asterisk (*): Matches any number of characters
          
          Example: "a * saved is a * earned"
          
          Question Mark (?): Matches single characters
          
          Example: wom?n finds "woman" or "women"
          
          Brackets []: Matches character sets
          
          Example: gr[ae]y finds "gray" or "grey"
